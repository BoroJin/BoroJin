### 🚀 Experiencia y Proyectos Destacados

🤖 **Ecosistema de Automatización Institucional (IPG) | *Analista de Automatización y Datos***
<br>Optimización del área financiera y de análisis institucional. Desarrollo de diversos bots en Python para la sincronización autónoma de datos con Google Sheets. Diseño de dashboards interactivos en Looker Studio para la toma de decisiones estratégicas basadas en datos en tiempo real.

🛡️ **Sentinel: Vigilancia y Seguimiento con IA | *Product Owner & FullStack Developer***
<br>Plataforma (Web/Mobile) de consultoría para la gestión de casos de acoso en entornos educativos y corporativos. Liderazgo del ciclo de vida del producto e integración de modelos de IA para la clasificación de emociones a través de texto y audio.

📍 **AIsys: Gestión de Incidencias con Computer Vision | *The Agent Hackathon***
<br>App móvil para el reporte inteligente de incidencias urbanas. Uso de Computer Vision para la validación automática de fotos y mapeo global de reportes, sumado a la implementación de feedback auditivo mediante la API de ElevenLabs.

🚛 **CamióGO: Arquitectura Backend Multitenant | *Desarrollador Backend***
<br>Plataforma logística diseñada para conectar desafíos del sector con soluciones de mercado. Implementación de una arquitectura multitenant con aislamiento de datos por subdominios y esquemas de base de datos independientes, reforzada con un diseño de sistema de permisos y roles (RBAC).
